Speech Transcriber & Translator

A web application built with Django that allows users to transcribe speech from audio files and translate the transcribed text into multiple languages. This project integrates speech recognition and translation APIs to provide a seamless experience.

Features

Upload audio files (e.g., .wav, .mp3) for transcription.

Convert spoken words into text using speech recognition.

Translate the transcribed text into multiple languages.

User-friendly web interface with a simple upload and results view.

Supports multiple languages for translation.

Tech Stack

Backend: Django, Python

Frontend: HTML, CSS

APIs/Libraries:

Whisper model api for speech transcribing

googletrans or any other translation API for translating text
